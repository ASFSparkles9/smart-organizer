# Умный локальный органайзер с ИИ

Интеллектуальное приложение для организации личной жизни и совместной работы с локальной обработкой данных.

## Основные возможности

### 📝 Личные заметки
- Текстовые заметки с форматированием
- Голосовой ввод
- Прикрепление фото и файлов
- Организация по папкам и тегам
- Поиск по содержимому
- Автоматическое создание задач из заметок

### 📅 Календарь
- Просмотр: день, неделя, месяц
- Drag & Drop для перемещения событий
- Цветовые категории
- Напоминания
- Умное позиционирование задач
- Плавающие задачи
- История жизни
- Персональные циклы (смены работы, менструальный цикл, сон, привычки)

### 👥 Совместная работа
- Общие события
- Перекрытие расписаний
- Автоматическое предложение времени
- Совместные заметки с ИИ-резюме
- Анализ свободных окон
- Стыковка местоположений (гео + пробки + транспорт)

### 🤖 Локальный ИИ
- Анализ загруженности
- Прогноз энергии и продуктивности
- Автоматическое создание задач
- Антипрокрастинационный режим
- Генерация сводок по дню/неделе
- Рекомендации по оптимизации баланса

### ⚖️ Баланс жизненных сфер
- Визуализация нагрузки по категориям
- Перераспределение времени
- Учет продуктивности

### 🏆 Мотивация
- Достижения и челленджи
- Графики продуктивности
- Мотивирующие напоминания

## Технологии

- **Frontend**: Next.js 14, React 18, TypeScript
- **Стилизация**: Tailwind CSS
- **База данных**: IndexedDB (Dexie.js)
- **Календарь**: react-big-calendar
- **Drag & Drop**: react-dnd
- **Графики**: Recharts
- **Анимации**: Framer Motion

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу [http://localhost:3000](http://localhost:3000)

## Архитектура

```
src/
├── app/              # Next.js App Router
├── components/       # Переиспользуемые компоненты
├── modules/          # Модули приложения
│   ├── notes/        # Личные заметки
│   ├── calendar/     # Календарь
│   ├── collaboration/# Совместная работа
│   ├── ai/           # Локальный ИИ
│   ├── balance/      # Баланс жизненных сфер
│   └── motivation/   # Мотивация
├── lib/              # Утилиты и хелперы
├── types/            # TypeScript типы
├── hooks/            # React хуки
└── store/            # Управление состоянием
```

## Особенности

- 🔒 **Полная локальность**: Все данные хранятся и обрабатываются локально
- 🚀 **Оффлайн-работа**: Приложение работает без интернета
- 🤖 **ИИ-анализ**: Умные рекомендации без передачи данных в облако
- 🎨 **Современный UI**: Интуитивный и красивый интерфейс
