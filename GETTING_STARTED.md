# Начало работы

## Установка зависимостей

```bash
npm install
```

## Запуск в режиме разработки

```bash
npm run dev
```

Приложение будет доступно по адресу [http://localhost:3000](http://localhost:3000)

## Сборка для продакшена

```bash
npm run build
npm start
```

## Структура данных

При первом запуске приложение автоматически создаст базу данных IndexedDB в браузере. Все данные хранятся локально и не передаются в облако.

## Основные функции

### 1. Заметки
- Создавайте заметки с текстом, тегами и папками
- Автоматическое извлечение задач из заметок
- Поиск по содержимому

### 2. Календарь
- Переключайтесь между видами: день, неделя, месяц
- Перетаскивайте события для изменения времени
- Создавайте персональные циклы (смены работы, менструальный цикл, сон, привычки)

### 3. ИИ Анализ
- Просматривайте анализ загруженности и продуктивности
- Получайте рекомендации по оптимизации
- Отслеживайте риск прокрастинации

### 4. Баланс жизненных сфер
- Визуализируйте распределение времени
- Сравнивайте с целями
- Оптимизируйте баланс работы, отдыха и здоровья

### 5. Мотивация
- Отслеживайте достижения
- Участвуйте в челленджах
- Просматривайте графики продуктивности

## Настройка

Настройки приложения хранятся в базе данных и включают:
- Тему (светлая/темная/авто)
- Язык интерфейса
- Часовой пояс
- Включение/выключение ИИ

## Резервное копирование

Для резервного копирования данных можно экспортировать IndexedDB через DevTools браузера или добавить функцию экспорта в приложение.

## Устранение неполадок

### База данных не инициализируется
- Убедитесь, что используете современный браузер с поддержкой IndexedDB
- Проверьте консоль браузера на наличие ошибок

### Стили не применяются
- Убедитесь, что Tailwind CSS правильно настроен
- Проверьте `tailwind.config.js`

### Календарь не отображается
- Проверьте, что `moment` установлен
- Убедитесь, что стили `react-big-calendar` импортированы

## Разработка

### Добавление нового модуля

1. Создайте папку в `src/modules/`
2. Создайте главный компонент модуля
3. Добавьте модуль в `src/app/page.tsx`
4. Добавьте пункт меню в `Sidebar.tsx`

### Добавление новых типов

Добавьте типы в `src/types/index.ts` и обновите схему базы данных в `src/lib/db.ts`.

### Расширение ИИ

Добавьте новые методы в класс `LocalAI` в `src/lib/ai/localAI.ts` или создайте новый класс, наследуя от `LocalAI`.
