# Архитектура приложения

## Обзор

Приложение построено на Next.js 14 с использованием App Router, React 18 и TypeScript. Все данные хранятся локально в IndexedDB через Dexie.js.

## Структура проекта

```
src/

│   ├── layout.tsx          # Корневой layout
│   ├── page.tsx            # Главная страница
│   └── globals.css         # Глобальные стили
├── components/             # Переиспользуемые компоненты
│   ├── layout/             # Компоненты макета
│   │   ├── Sidebar.tsx     # Боковая панель навигации

│   ├── balance/            # Баланс жизненных сфер
```

## База данных

### IndexedDB через Dexie.js

База данных `SmartOrganizerDB` содержит следующие таблицы:

- **notes** - Заметки пользователя
- **folders** - Папки для организации заметок
- **events** - События календаря
- **tasks** - Задачи
- **cycles** - Персональные циклы
- **balance** - Данные о балансе жизненных сфер
- **achievements** - Достижения
- **challenges** - Челленджи
- **collaborations** - Совместные события
- **settings** - Настройки пользователя

Все данные хранятся локально, без синхронизации с облаком.

## Архитектура локального ИИ

### Текущая реализация

Класс `LocalAI` в `src/lib/ai/localAI.ts` предоставляет следующие функции:

1. **Анализ загруженности** (`analyzeWorkload`)
   - Вычисляет процент загруженности на основе событий календаря
   - Учитывает продолжительность событий
   - Максимальная загруженность - 8 часов в день

2. **Прогноз энергии** (`forecastEnergy`)
   - Генерирует прогноз на 7 дней вперед
   - Учитывает персональные циклы (менструальный, сон)
   - Рекомендует оптимальные типы задач

3. **Извлечение задач из заметок** (`extractTasksFromNote`)
   - Использует паттерны регулярных выражений
   - Ищет ключевые слова: "нужно", "надо", "todo"
   - Возвращает список извлеченных задач

4. **Оценка риска прокрастинации** (`assessProcrastinationRisk`)
   - Анализирует просроченные задачи
   - Учитывает количество задач с высоким приоритетом
   - Оценивает загруженность

5. **Генерация сводок** (`generateDailySummary`)
   - Создает краткую сводку по дню
   - Анализирует выполненные задачи
   - Дает рекомендации

6. **Рекомендации по балансу** (`getBalanceRecommendations`)
   - Анализирует распределение времени по категориям
   - Предлагает оптимизации

### Будущие улучшения

Для более продвинутого ИИ можно интегрировать:

1. **WebLLM** - Локальные языковые модели в браузере
2. **Transformers.js** - Запуск моделей Transformers в браузере
3. **ONNX Runtime Web** - Выполнение ONNX моделей

Пример интеграции WebLLM:

```typescript
import { LLM } from '@mlc-ai/web-llm';

const llm = new LLM({
  model: 'Llama-2-7b-chat-hf-q4f32_1',
  initProgressCallback: (progress) => {
    console.log('Загрузка модели:', progress);
  }
});

async function analyzeWithLLM(text: string) {
  const response = await llm.generate(text);
  return response;
}
```

## Модули приложения

### 1. Личные заметки (`modules/notes/`)

- **NotesModule** - Главный компонент модуля
- **NoteEditor** - Редактор заметок
- **NoteList** - Список заметок
- **FolderSidebar** - Боковая панель с папками

Возможности:
- Создание, редактирование, удаление заметок
- Организация по папкам и тегам
- Поиск по содержимому
- Автоматическое извлечение задач (через ИИ)

### 2. Календарь (`modules/calendar/`)

- **CalendarModule** - Главный компонент
- **CalendarView** - Визуализация календаря (react-big-calendar)
- **EventEditor** - Редактор событий
- **CyclesPanel** - Панель персональных циклов
- **useDragAndDrop** - Хук для drag & drop

Возможности:
- Просмотр: день, неделя, месяц
- Drag & Drop для перемещения событий
- Цветовые категории
- Напоминания
- Плавающие задачи
- Персональные циклы

### 3. Совместная работа (`modules/collaboration/`)

- **CollaborationModule** - Главный компонент

Будущие возможности:
- Общие события
- Перекрытие расписаний
- Автоматическое предложение времени
- Совместные заметки с ИИ-резюме
- Анализ свободных окон
- Стыковка местоположений

### 4. Локальный ИИ (`modules/ai/`)

- **AIModule** - Визуализация анализа ИИ

Отображает:
- Загруженность
- Продуктивность
- Риск прокрастинации
- Прогноз энергии
- Рекомендации

### 5. Баланс жизненных сфер (`modules/balance/`)

- **BalanceModule** - Визуализация баланса

Возможности:
- Круговая диаграмма распределения времени
- Сравнение с целями
- Детальная статистика по категориям

### 6. Мотивация (`modules/motivation/`)

- **MotivationModule** - Система мотивации

Возможности:
- Достижения
- Челленджи
- Графики продуктивности

## Стилизация

Приложение использует Tailwind CSS с кастомной темой:

- Цветовая палитра на основе primary (синий)
- Поддержка темной темы
- Адаптивный дизайн
- Анимации через Framer Motion

## Производительность

- Все данные хранятся локально (IndexedDB)
- Минимальные зависимости
- Оптимизация через Next.js
- Ленивая загрузка модулей

## Безопасность

- Нет передачи данных в облако
- Все обработка локально
- Приватность гарантирована
- Нет внешних API запросов
